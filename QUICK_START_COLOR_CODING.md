# Быстрый старт - Цветовая индикация кодов ТНВЭД

## 🎯 Что это?

Система автоматически раскрашивает коды ТНВЭД в Excel-файлах:
- 🟢 **Зеленый** - высокая уверенность (≥ 0.185)
- 🔴 **Красный** - низкая уверенность (< 0.185), требуется проверка
- ⚪ **Белый** - найдено по URL (= 1.0), максимальная надежность

## 🚀 Как использовать?

### 1. Обработайте файл как обычно

```bash
# Через веб-интерфейс
http://localhost:8000

# Или через командную строку
python start_batch_processor.py --input file.xlsx --output result.xlsx
```

### 2. Откройте результат

Откройте `result.xlsx` в Excel - коды уже раскрашены!

### 3. Проверьте красные ячейки

Найдите все 🔴 красные ячейки и проверьте коды вручную.

## 📊 Пример

### До обработки
```
| Product Description      |
|--------------------------|
| Кофе арабика 1кг         |
| Неизвестный товар        |
```

### После обработки
```
| Product Description | TNVED_Code | Цвет |
|---------------------|------------|------|
| Кофе арабика 1кг    | 0901110000 | 🟢   |
| Неизвестный товар   | 9999999999 | 🔴   |
```

## ✅ Что делать дальше?

1. **Зеленые ячейки** → Использовать сразу ✅
2. **Красные ячейки** → Проверить и исправить ⚠️
3. **Белые ячейки** → Использовать сразу ✅

## ❓ Вопросы?

**Q: Почему порог 0.185?**  
A: Оптимальное значение на основе анализа качества подбора.

**Q: Можно ли изменить цвета?**  
A: Да, но требуется изменение кода. См. техническую документацию.

**Q: Цвета не отображаются?**  
A: Убедитесь, что файл открыт в Excel 2007 или выше.
