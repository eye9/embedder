### Технические детали

#### Правила цветовой индикации

| Условие | Цвет | RGB | Значение |
|---------|------|-----|----------|
| Score = 1.0 | Белый (без цвета) | - | URL match, максимальная надежность |
| Score ≥ 0.185 | Зеленый | 00FF00 | Высокая уверенность |
| Score < 0.185 | Красный | FF0000 | Низкая уверенность, требуется проверка |

#### Зависимости
- `openpyxl` - для работы с Excel и применения стилей
- `pandas` - для обработки данных

#### Обратная совместимость
- ✅ Полная обратная совместимость с существующим кодом
- ✅ Автоматическое применение цветовой индикации
- ✅ Graceful degradation при ошибках (файл создается без цветов)

### Использование

#### Автоматическое применение
```python
from batch_processor.services.excel_processor import ExcelProcessor

processor = ExcelProcessor()
processor.write_results(
    original_file=input_file,
    results=processing_results,
    output_file=output_file
)
# Цвета применяются автоматически
```

#### Тестирование
```bash
python test_color_coding.py
```

### Примеры

#### Пример 1: Высокая уверенность (зеленый)
```python
{
    'row_index': 0,
    'tnved_code': '0901110000',
    'selection_reason': 'Similarity Score: 0.850',
    'confidence_score': 0.850  # → Зеленый
}
```

#### Пример 2: Низкая уверенность (красный)
```python
{
    'row_index': 1,
    'tnved_code': '9999999999',
    'selection_reason': 'Similarity Score: 0.120',
    'confidence_score': 0.120  # → Красный
}
```

#### Пример 3: URL match (белый)
```python
{
    'row_index': 2,
    'tnved_code': '1234567890',
    'selection_reason': 'Found by URL: https://ozon.ru/...',
    'confidence_score': 1.0  # → Белый (без цвета)
}
```

### Преимущества

1. **Визуальная оценка качества** - Быстрая идентификация надежных и сомнительных кодов
2. **Упрощение проверки** - Легко найти коды, требующие ручной проверки
3. **Повышение доверия** - Пользователи видят уровень уверенности системы
4. **Экономия времени** - Не нужно проверять все коды, только красные

### Известные ограничения

1. Цветовая индикация применяется только к колонке `TNVED_Code`
2. Требуется Excel 2007 или выше для корректного отображения
3. Цвета статичны и не обновляются при изменении данных
