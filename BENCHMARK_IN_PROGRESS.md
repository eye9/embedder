# 🔄 Тестирование моделей в процессе

## Что происходит

Запущен автоматизированный бенчмарк всех моделей эмбеддингов.

### Процесс

Для каждой модели:
1. ✅ Загрузка модели
2. 🔄 Создание отдельной базы данных
3. 🔄 Индексация 1000 записей из `tnved_1-1000.xlsx`
4. 🔄 Тестирование поиска (10 запросов)
5. 🔄 Сбор метрик

### Модели для тестирования

1. **FRIDA** (~3.1GB) - 🔄 В процессе
2. **E5-Small** (~470MB) - ⏳ Ожидание
3. **MiniLM-L12** (~470MB) - ⏳ Ожидание
4. **RuBERT-Tiny2** (~120MB) - ⏳ Ожидание

### Примерное время

- **FRIDA**: ~10-15 минут (самая медленная)
- **E5-Small**: ~3-5 минут
- **MiniLM-L12**: ~3-5 минут
- **RuBERT-Tiny2**: ~2-3 минуты

**Общее время**: ~20-30 минут

### Что будет в отчете

- ⚡ Скорость загрузки модели
- 📊 Скорость индексации данных
- 🔍 Скорость поиска
- 🎯 Качество результатов (similarity scores)
- 📝 Примеры результатов поиска
- 💡 Рекомендации по выбору модели

### Результаты

Отчет будет сохранен в: **`docs/BENCHMARK_REPORT.md`**

### Прогресс

Проверить прогресс можно командой:
```bash
# В PowerShell
Get-Content benchmark_dbs\benchmark_frida\chroma.sqlite3 -ErrorAction SilentlyContinue
```

Или просто подождите - скрипт сам выведет итоговый отчет!
