# Цветовая индикация кодов ТНВЭД

## Что это?

При обработке Excel-файлов система автоматически раскрашивает ячейки с кодами ТНВЭД в зависимости от уровня уверенности в подобранном коде.

## Как это работает?

### 🟢 Зеленый цвет
**Когда:** Similarity Score ≥ 0.185

**Что это значит:** Код подобран с высокой уверенностью. Можно использовать без дополнительной проверки.

**Примеры:**
- Score: 0.850 → 🟢 Очень высокая уверенность
- Score: 0.250 → 🟢 Хорошая уверенность
- Score: 0.185 → 🟢 Приемлемая уверенность

### 🔴 Красный цвет
**Когда:** Similarity Score < 0.185

**Что это значит:** Код подобран с низкой уверенностью. **Требуется ручная проверка!**

**Примеры:**
- Score: 0.120 → 🔴 Низкая уверенность
- Score: 0.050 → 🔴 Очень низкая уверенность

**Рекомендация:** Проверьте описание товара и подобранный код вручную. Возможно, потребуется выбрать другой код.

### ⚪ Без цвета (белый)
**Когда:** Similarity Score = 1.0 (код найден по URL)

**Что это значит:** Код найден точным совпадением по URL в базе подтвержденных кодов. **Максимальная надежность!**

**Пример:**
- Товар с Ozon/Wildberries → ⚪ Код из базы подтвержденных

## Как использовать?

### 1. Загрузите файл
Загрузите Excel-файл через веб-интерфейс или API.

### 2. Дождитесь обработки
Система автоматически обработает все строки.

### 3. Скачайте результат
Скачайте обработанный файл с цветовой индикацией.

### 4. Проверьте результаты

#### ✅ Зеленые ячейки
- Можно использовать сразу
- Высокая уверенность в коде

#### ⚠️ Красные ячейки
- Требуют проверки
- Низкая уверенность в коде
- Проверьте описание товара
- При необходимости исправьте код вручную

#### ✨ Белые ячейки
- Максимальная надежность
- Код из базы подтвержденных
- Можно использовать без проверки

## Пример

### Исходный файл

| Product Detailed Description | 
|------------------------------|
| Кофе в зернах арабика 1кг    |
| Неизвестный товар            |
| Товар с Ozon                 |

### Результат с цветами

| Product Detailed Description | TNVED_Code | Selection_Reason | Цвет |
|------------------------------|------------|------------------|------|
| Кофе в зернах арабика 1кг    | 0901110000 | Similarity Score: 0.850 | 🟢 |
| Неизвестный товар            | 9999999999 | Similarity Score: 0.120 | 🔴 |
| Товар с Ozon                 | 1234567890 | Found by URL: https://ozon.ru/... | ⚪ |

### Что делать дальше?

1. **Зеленая строка (кофе)** → Использовать код 0901110000 ✅
2. **Красная строка (неизвестный товар)** → Проверить и исправить код ⚠️
3. **Белая строка (Ozon)** → Использовать код 1234567890 ✅

## Часто задаваемые вопросы

### Почему порог именно 0.185?

Это значение выбрано на основе анализа качества подбора кодов. Оно обеспечивает хороший баланс между точностью и полнотой результатов.

### Можно ли изменить порог?

Да, но это требует изменения кода. Обратитесь к разработчику или см. [техническую документацию](COLOR_CODING_GUIDE.md).

### Что делать с красными ячейками?

1. Откройте файл в Excel
2. Найдите все красные ячейки
3. Проверьте описание товара
4. Найдите правильный код ТНВЭД вручную
5. Исправьте код в ячейке

### Цвета не отображаются

Убедитесь, что:
- Файл открыт в Excel или совместимой программе
- Файл не поврежден
- Используется актуальная версия системы

### Можно ли отключить цветовую индикацию?

Нет, цветовая индикация применяется автоматически. Но вы можете удалить форматирование в Excel:
1. Выделите колонку TNVED_Code
2. Правая кнопка мыши → Формат ячеек
3. Заливка → Нет заливки
