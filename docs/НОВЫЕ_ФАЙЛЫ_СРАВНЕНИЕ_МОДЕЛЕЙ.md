# Новые файлы: Сравнение моделей эмбеддингов

## 📊 Скрипты для сравнения (3 файла)

### 1. `test_single_model.py`
**Быстрый тест одной модели**
- Загрузка и проверка модели
- Измерение скорости генерации эмбеддингов
- Тест поиска с примерами результатов
- Использование: `python test_single_model.py <model_name> [device]`

### 2. `compare_models.py`
**Полное сравнение всех моделей**
- Сравнение 4 моделей: FRIDA, E5-Small, MiniLM-L12, RuBERT-Tiny2
- Метрики: скорость загрузки, генерации, поиска, качество
- Итоговая таблица с рекомендациями
- Использование: `python compare_models.py`

### 3. `compare_search_results.py`
**Детальное сравнение результатов поиска**
- Side-by-side сравнение результатов для одинаковых запросов
- Анализ пересечений между моделями
- Сравнение similarity scores
- Использование: `python compare_search_results.py`

## 🚀 Быстрый старт

### 1. Установите зависимости
```bash
pip install tabulate
```

### 2. Протестируйте E5-Small
```bash
# Для GTX 1060 3GB
python test_single_model.py intfloat/multilingual-e5-small cuda

# Для CPU
python test_single_model.py intfloat/multilingual-e5-small
```

### 3. Если понравилось, обновите config.yaml
```yaml
model:
  name: "intfloat/multilingual-e5-small"
  device: "cuda"
processing:
  batch_size: 32
```

### 4. Пересоздайте базу данных
```bash
python load_tnved.py
```

## 💡 Рекомендация для вашей системы

**GTX 1060 3GB + CUDA:**
```yaml
model:
  name: "intfloat/multilingual-e5-small"
  device: "cuda"
processing:
  batch_size: 32
```

**Преимущества:**
- ✅ В 6.5 раз меньше памяти (470 MB vs 3.1 GB)
- ✅ В 3-5 раз быстрее
- ✅ Отличное качество для русского языка
- ✅ Стабильная работа на 3GB GPU

## ⚠️ Важно

**При смене модели обязательно пересоздайте базу данных:**
```bash
python load_tnved.py
```

Разные модели создают эмбеддинги разной размерности, поэтому старая база данных не будет совместима с новой моделью.
