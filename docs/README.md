## 📖 Описание документов

### Сравнение моделей

#### MODEL_ALTERNATIVES_SUMMARY.md
Краткое резюме с тремя альтернативами FRIDA:
- E5-Small (рекомендуется)
- RuBERT-Tiny2 (самая быстрая)
- MiniLM-L12 (проверенная)

Включает быстрый старт и рекомендации по выбору.

#### QUICK_MODEL_COMPARISON.md
Быстрый старт по сравнению моделей:
- Описание каждой модели с конфигурацией
- Команды для запуска тестов
- Что выбрать для разных сценариев
- Ожидаемые результаты

#### MODEL_COMPARISON_GUIDE.md
Полное руководство по сравнению моделей:
- Детальное описание всех моделей
- Метрики сравнения
- Интерпретация результатов
- Переключение между моделями
- Оптимизация производительности

#### ВАЖНО_РАЗМЕРНОСТЬ_ЭМБЕДДИНГОВ.md
Объяснение проблемы размерности эмбеддингов:
- Почему база несовместима с разными моделями
- Как решить проблему
- Рекомендуемый workflow

#### FAQ_СРАВНЕНИЕ_МОДЕЛЕЙ.md
Часто задаваемые вопросы:
- Общие вопросы
- Технические вопросы
- Проблемы и решения
- Производительность

#### MODELS_VISUAL_COMPARISON.txt
Визуальное сравнение моделей в виде ASCII-таблиц и диаграмм.

#### COMMANDS_CHEATSHEET.txt
Шпаргалка всех команд для:
- Установки зависимостей
- Тестирования моделей
- Смены модели
- Проверки CUDA

## 🔗 Связанные ресурсы

### Скрипты тестирования
Скрипты для тестирования находятся в папке `../benchmarks/`:
- `test_single_model.py` - Тест одной модели
- `compare_models.py` - Сравнение всех моделей
- `compare_search_results.py` - Сравнение результатов поиска
- `compare_prefix_impact.py` - Сравнение влияния префиксов

### Конфигурация
- `../config.yaml` - Главный конфигурационный файл

### Основной код
- `../services/` - Сервисы системы
- `../models/` - Модели данных
- `../utils/` - Утилиты
