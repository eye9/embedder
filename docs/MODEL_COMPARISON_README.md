# Сравнение моделей эмбеддингов

## Что добавлено

Система для сравнения различных моделей эмбеддингов для поиска по ТНВЭД.

## Новые файлы

### Скрипты
- **`compare_models.py`** - Общее сравнение производительности моделей
- **`compare_search_results.py`** - Детальное сравнение результатов поиска

### Документация
- **`MODEL_COMPARISON_GUIDE.md`** - Полное руководство по сравнению
- **`QUICK_MODEL_COMPARISON.md`** - Быстрый старт и рекомендации

## Сравниваемые модели

1. **FRIDA** (текущая) - 3.1 GB, медленная, высокое качество
2. **E5-Small** (рекомендуется) - 470 MB, быстрая, отличное качество
3. **MiniLM-L12** - 470 MB, надежная, хороший баланс
4. **RuBERT-Tiny2** - 120 MB, очень быстрая, компактная

## Использование

### Быстрое сравнение
```bash
# Установить зависимости
pip install tabulate

# Запустить общее сравнение
python compare_models.py

# Детальное сравнение результатов
python compare_search_results.py
```

### Смена модели

1. Отредактируйте `config.yaml`:
```yaml
model:
  name: "intfloat/multilingual-e5-small"  # Новая модель
  device: "cuda"  # или "cpu"
```

2. Пересоздайте базу данных:
```bash
python load_tnved.py
```

3. Протестируйте:
```bash
python search_tnved.py
```

## Рекомендации

### Для вашей GTX 1060 3GB
Используйте **E5-Small** с CUDA:
```yaml
model:
  name: "intfloat/multilingual-e5-small"
  device: "cuda"
processing:
  batch_size: 32
```

**Преимущества:**
- В 6.5 раз меньше памяти (470 MB vs 3.1 GB)
- В 3-5 раз быстрее
- Отличное качество для русского языка
- Стабильно работает на 3GB GPU

### Для CPU-only
Используйте **RuBERT-Tiny2**:
```yaml
model:
  name: "cointegrated/rubert-tiny2"
  device: "cpu"
processing:
  batch_size: 100
```

## Метрики сравнения

Скрипты измеряют:
- ⏱️ Время загрузки модели
- 📊 Размерность эмбеддингов
- 🚀 Скорость генерации (эмбеддингов/сек)
- 🔍 Скорость поиска (секунды)
- 🎯 Качество поиска (средний similarity score)

## Важно! ⚠️

**При смене модели обязательно пересоздайте базу данных!**

Разные модели создают эмбеддинги разной размерности, поэтому старая база данных не будет совместима с новой моделью.
