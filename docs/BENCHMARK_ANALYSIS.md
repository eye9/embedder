## Рекомендации

### Для вашей GTX 1060 3GB + CUDA

**Рекомендуется: E5-Small**

```yaml
model:
  name: "intfloat/multilingual-e5-small"
  device: "cuda"
processing:
  batch_size: 32
```

**Почему:**
- Лучшее качество результатов
- Отличная скорость
- Компактный размер (поместится в GPU)
- Универсальная для разных задач

**Ожидаемая производительность на CUDA:**
- Индексация: ~200-300 зап/с (в 2-3 раза быстрее CPU)
- Поиск: ~0.005-0.010с

---

### Для CPU-only систем

**Рекомендуется: RuBERT-Tiny2**

```yaml
model:
  name: "cointegrated/rubert-tiny2"
  device: "cpu"
processing:
  batch_size: 100
```

**Почему:**
- Самая быстрая на CPU
- Минимальные требования
- Хорошее качество
- Идеально для real-time

---

### Для максимального качества

**Рекомендуется: E5-Small**

E5-Small показала лучшее качество среди всех моделей.
