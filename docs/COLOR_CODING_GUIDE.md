# Руководство по цветовой индикации кодов ТНВЭД

## Обзор

Система автоматически раскрашивает ячейки с кодами ТНВЭД в выходном Excel-файле в зависимости от уровня уверенности (similarity score) подбора кода.

## Правила раскраски

### 🟢 Зеленый цвет (RGB: 00FF00)
**Условие:** `SIMILARITY SCORE >= 0.185`

Зеленый цвет указывает на **высокую уверенность** в подобранном коде. Код был найден с помощью семантического поиска, и степень схожести превышает пороговое значение.

**Примеры:**
- Score: 0.850 → Зеленый (очень высокая уверенность)
- Score: 0.250 → Зеленый (выше порога)
- Score: 0.185 → Зеленый (на пороге)

### 🔴 Красный цвет (RGB: FF0000)
**Условие:** `SIMILARITY SCORE < 0.185`

Красный цвет указывает на **низкую уверенность** в подобранном коде. Рекомендуется **ручная проверка** таких кодов.

**Примеры:**
- Score: 0.120 → Красный (низкая уверенность)
- Score: 0.050 → Красный (очень низкая уверенность)

### ⚪ Без цвета (белый/по умолчанию)
**Условие:** `SIMILARITY SCORE = 1.0` (код подобран по URL)

Отсутствие цвета указывает на то, что код был найден **точным совпадением по URL** в базе данных подтвержденных кодов. Это самый надежный способ подбора кода.

**Пример:**
- Score: 1.0 → Без цвета (найдено по URL)

## Технические детали

### Реализация

Цветовая индикация реализована в следующих модулях:

1. **`batch_processor/services/excel_processor.py`**
   - Метод `write_results()` - записывает результаты и применяет цвета
   - Метод `_apply_color_coding()` - применяет цветовую схему к ячейкам

2. **`batch_processor/services/enhanced_excel_processor.py`**
   - Метод `write_hybrid_results()` - поддерживает цветовую индикацию для гибридной обработки

### Структура данных

Цветовая индикация основана на поле `confidence_score` в результатах обработки:

```python
{
    'row_index': 0,
    'tnved_code': '0901110000',
    'selection_reason': 'Found by URL match',
    'confidence_score': 1.0  # Используется для определения цвета
}
```

### Алгоритм применения цветов

```python
if confidence_score == 1.0:
    # URL match - no color (leave default)
    pass
elif confidence_score >= 0.185:
    # High confidence - green
    cell.fill = green_fill
else:
    # Low confidence - red
    cell.fill = red_fill
```

## Использование

### Автоматическое применение

Цветовая индикация применяется автоматически при обработке файлов через:

1. **Batch Processor Web Interface**
   - Загрузите файл через веб-интерфейс
   - Выберите режим обработки
   - Скачайте результат с цветовой индикацией

2. **API**
   ```python
   from batch_processor.services.excel_processor import ExcelProcessor
   
   processor = ExcelProcessor()
   processor.write_results(
       original_file=input_file,
       results=processing_results,
       output_file=output_file
   )
   # Цвета применяются автоматически
   ```

3. **Command Line**
   ```bash
   python start_batch_processor.py --input file.xlsx --output result.xlsx
   ```

### Интерпретация результатов

При открытии обработанного файла:

1. **Зеленые ячейки** - коды можно использовать с высокой уверенностью
2. **Красные ячейки** - требуется ручная проверка и возможная корректировка
3. **Белые ячейки** - коды найдены точным совпадением, максимальная надежность

## Настройка порогового значения

Пороговое значение `0.185` установлено на основе анализа качества подбора кодов. Если требуется изменить это значение:

1. Откройте файл `batch_processor/services/excel_processor.py`
2. Найдите метод `_apply_color_coding()`
3. Измените условие:
   ```python
   elif confidence_score >= 0.185:  # Измените это значение
   ```

## Тестирование

Для проверки работы цветовой индикации используйте тестовый скрипт:

```bash
python test_color_coding.py
```

Скрипт создаст тестовый файл с различными значениями similarity score и применит цветовую индикацию.

## Ограничения

1. Цветовая индикация применяется только к колонке `TNVED_Code`
2. Требуется библиотека `openpyxl` для работы с Excel
3. Цвета применяются после записи данных в файл
4. При ошибке применения цветов файл все равно будет создан, но без цветовой индикации

## Устранение неполадок

### Цвета не применяются

**Проблема:** Файл создается, но ячейки не раскрашены

**Решение:**
1. Проверьте, что установлена библиотека `openpyxl`:
   ```bash
   pip install openpyxl
   ```
2. Проверьте логи на наличие предупреждений о цветовой индикации
3. Убедитесь, что в результатах присутствует поле `confidence_score`

### Неправильные цвета

**Проблема:** Цвета не соответствуют ожидаемым

**Решение:**
1. Проверьте значения `confidence_score` в колонке `Selection_Reason`
2. Убедитесь, что пороговое значение `0.185` корректно
3. Проверьте, что URL matches имеют score = 1.0

## Дополнительная информация

- Цветовая индикация не влияет на данные в файле
- Цвета видны только при открытии файла в Excel или совместимых программах
- Цветовая схема совместима с Excel 2007 и выше
